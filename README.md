# News Explorer Application

News explorer is a React-based web app that allows users to search for news articles by keyword using a public News API.
The app provides a clean, responsive interface and follows a structured design based on Figma specification.

## Features

- Search for news articles by keyword.
- Save favorite articles.
- Fully responsive design.

# Technologies Used

- React
- Javascript
- CSS
- REST API
- Figma
